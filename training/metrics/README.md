# Metrics Reports

This folder contains aggregated metrics generated by `/metrics` command.

## Report Types

| File Pattern | Description |
|--------------|-------------|
| `YYYY-MM-DD_weekly.md` | Weekly summary (last 7 days) |
| `YYYY-MM-DD_all-time.md` | Complete historical metrics |
| `YYYY-MM-DD_by-type.md` | Breakdown by response type |
| `YYYY-MM-DD_errors.md` | Error analysis report |
| `YYYY-MM-DD_cost.md` | Token usage and cost analysis |

## Usage

Run `/metrics` and select a report type to generate.

Reports are cumulative - old reports are preserved for historical comparison.

## Token Usage Data

Token usage is tracked in `training/usage/`:
- `YYYY-MM-DD.json` - Daily machine-readable data
- `YYYY-MM-DD.md` - Daily human-readable reports

See [training/usage/README.md](../usage/README.md) for details.
