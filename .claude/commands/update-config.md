# Update HeyReach Configuration

Update settings for HeyReach conversation processing.

## Instructions

1. Read `config.md` and show current values:
   - FETCH_LIMIT
   - MAX_MESSAGES_TO_SEND
   - LEAD_SAMPLE_1, LEAD_SAMPLE_2, LEAD_SAMPLE_3

2. Ask which setting to update

3. Update the value in `config.md`

4. Confirm the change

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| FETCH_LIMIT | 200 | Conversations to fetch per run |
| MAX_MESSAGES_TO_SEND | 10 | Messages to send per run |
| LEAD_SAMPLE_1 | (varies) | First lead sample |
| LEAD_SAMPLE_2 | (varies) | Second lead sample |
| LEAD_SAMPLE_3 | (varies) | Third lead sample |
